## 📖 插件介绍

美学提升插件是一个为 AI 绘画创作者设计的综合视觉参考工具，提供经典的构图法则、专业打光技巧和知名画师百科，帮助你快速提升作品的美学品质。

### ✨ 核心功能

- **📐 构图技巧**: 12 种经典构图示例（S 型、三角形、九宫格等）
- **💡 打光技巧**: 11 种专业光位演示（伦勃朗光、丁达尔光、逆光等）
- **🎨 画师百科**: 多风格知名画师作品和风格特点介绍
  - 支持点击图片放大查看
  - 画师名称和介绍可点击复制
  - 按风格分类浏览
- **🎬 AI 智能分析** (可选): Qwen3.5-VL 多模态大模型分析
  - 需要安装 Ollama 和 Qwen3.5 模型
  - 支持图片和视频分析
  - 自动识别构图、灯光、分镜等元素
- **🔍 点击放大**: 支持全屏查看高清细节
- **📱 响应式设计**: 完美适配各种屏幕尺寸
- **🎯 深色主题**: 优雅的深色界面，减少视觉疲劳

### 🎯 适用场景

- AI 绘画提示词参考
- 摄影构图学习
- 灯光布置灵感
- 美术基础训练
- 画师风格研究

## 🚀 安装方法

### 方式一：手动安装

1. 进入 WebUI 的 `extensions` 目录
2. 克隆或下载本插件到 `sd-webui-AestheticEnhancement-llama.cpp` 文件夹
3. 将ui_tempdir文件放入webui\modules目录
4. 重启 WebUI

### 方式二：使用 Git

```bash
cd your_webui_directory/extensions
git clone <repository_url> sd-webui-AestheticEnhancement-llama.cpp
```

### 方式三：直接下载

从 GitHub 或其他代码托管平台下载 ZIP 文件，解压到 `extensions` 目录。

## 💡 使用说明

### 启动插件

1. 启动 Stable Diffusion WebUI Forge
2. 在主界面顶部标签栏找到 **"🎨 美学提升"** 标签
3. 点击进入即可浏览所有素材

### 浏览构图技巧

- 切换到 **📐 构图技巧** 标签页
- 查看各种构图类型的示例图片
- 鼠标悬停可查看构图名称和简介
- **点击图片** 可全屏放大查看详情
- 展开底部 **📖 构图技巧详解** 阅读详细说明

### 浏览打光技巧

- 切换到 **💡 打光技巧** 标签页
- 查看不同光位的照明效果
- **点击图片** 可全屏放大查看细节
- 展开底部 **💡 打光技巧详解** 学习布光原则

### 浏览画师百科

- 切换到 **🎨 画师百科** 标签页
- 选择不同的画风分类标签
- **点击图片** 可放大查看画师作品
- **点击画师名称** 可复制名称到剪贴板
- **点击介绍文本** 可复制完整介绍
- 展开底部 **📖 画风特点详解** 了解风格特征

### 🎬 AI 智能分析（可选功能）

**注意**: 此功能需要额外配置 Ollama 和 Qwen3.5 模型

#### 前置要求

1. **安装 Ollama**: 从 https://ollama.com 下载并安装
2. **安装 Qwen3.5 模型**: 运行命令 `ollama run qwen3.5:4b`
3. **安装 Python 依赖**: `pip install requests opencv-python`
4. **启动 Ollama 服务**: 确保 Ollama 在后台运行

#### 使用步骤

1. 切换到 **🎬 AI 智能分析** 标签页
2. 点击"🔌 测试 Ollama 连接"确认服务正常
3. 选择分析模式（图片/视频）
4. 选择分析类型：
   - 🔍 综合分析
   - 📐 仅构图分析
   - 💡 仅灯光分析
   - 🎥 仅分镜分析
5. 上传图片或视频
6. 点击"🚀 开始分析"

#### 功能特点

- **图片分析**: 自动识别构图、灯光、色彩、镜头语言
- **视频分析**: 智能镜头检测，关键帧提取，批量分析
- **多模型支持**: qwen3.5:2b/4b/9b，平衡速度与质量
- **完全免费**: 本地运行，无需 API Key

### 使用技巧

1. **提示词参考**: 
   - 构图：`rule of thirds composition`, `symmetrical composition`
   - 灯光：`rembrandt lighting`, `backlight`, `volumetric lighting`
   - 画师风格：在提示词中加入画师名称，如 `by anmi style`

2. **对比学习**: 对比不同构图和光位的视觉效果，理解其情感表达差异

3. **实践应用**: 在创作时有意识地运用学到的构图、布光技巧和画师风格

## ❓ 常见问题

### Q: 图片无法加载？

A: 请检查以下事项：
1. 确认 `Aesthetic-Enhancement` 目录存在于插件文件夹中
2. 确认目录下有对应的素材子目录
3. 检查图片文件是否完整（PNG/JPG 格式）

### Q: 点击卡片没有反应？

A: 可能是 JavaScript 未正确加载，尝试：
1. 刷新浏览器页面（Ctrl+F5 强制刷新）
2. 清除浏览器缓存
3. 检查浏览器控制台是否有报错（F12）

### Q: 画师百科显示重复内容？

A: 这是 Gradio 组件状态共享问题，已通过纯 HTML 实现解决。如果仍然出现，尝试：
1. 完全重启 WebUI
2. 清除浏览器缓存

### Q: 如何添加自定义素材？

A: 可以！直接将你的参考图片放入对应的素材目录即可：
- 构图图片放入 `Aesthetic-Enhancement/构图技巧/`
- 打光图片放入 `Aesthetic-Enhancement/打光技巧/`
- 画师素材放入 `Aesthetic-Enhancement/画师百科/[风格]/` 目录
  - 图片文件：`[画师名].jpg`
  - 说明文件：`[画师名].txt`

### Q: 如何贡献新的画师素材？

A: 欢迎贡献！请按照以下格式组织素材：
1. 在 `Aesthetic-Enhancement/画师百科/` 下创建风格子目录
2. 为每个画师添加：
   - 一张代表作品图片（建议 800x1200 像素）
   - 一个 TXT 文件，包含风格特点介绍
3. 提交 Pull Request 或联系开发者
